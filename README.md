> **"La precisión institucional es la única respuesta válida ante la fuerza bruta."**

SLINGSHOT es una **Terminal Algorítmica Local-First** diseñada para identificar la huella del Smart Money en tiempo real. Utiliza una arquitectura de baja latencia centrada en el análisis de liquidez, bloques de órdenes y divergencias de mercado.

## 🏗️ 2. Arquitectura & Stack
- **Engine (Back):** Python 3.12 (OHLCV Processing) + FastAPI (Streaming WS).
- **Core (RAM):** MemoryStore centralizado (Zero-Latency Local Compute).
- **Strategy (SMC):** KillZones + Liquidity Sweeps + POI (OB/FVG) + SMT Divergence.
- **Frontend (UI):** Next.js 15 (React 19) + Tailwind 4 + Zustand.

## 🚀 3. Inicio Rápido (One-Click)
1. Asegúrate de tener instalado Python 3.12+ y Node.js.
2. Configura tu `.env` (usa `.env.example` como plantilla).
3. **Lanzamiento:** Ejecuta el script `./start.ps1` en PowerShell.
